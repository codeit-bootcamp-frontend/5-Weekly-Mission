##작업정리

커다란 부분부터 구현하고 작은 부분으로 들어간다.

1. html

   \*최대한 깔끔하고 명확하게 페이지를 구성한다. (적절한 시멘틱 태그의 활용)

   \*기본적으로 루트페이지에서 a태그를 통해 링크를 연결해준다.

   \*이미지들은 따로 폴더로 정리한다.

2. css

   \*요구사항에 맞춰 페이지에 스타일을 적용한다.
